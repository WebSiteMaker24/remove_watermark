README – Masquage du watermark Windows
Description

Ce script permet de masquer le watermark Windows (texte en bas à droite du bureau) en utilisant un réglage d’affichage via le registre utilisateur.
Il n’active pas Windows, ne modifie aucun fichier système, et est totalement réversible.

Prérequis

Windows 10 / Windows 11

Compte utilisateur avec droits administrateur

Utilisation

Clic droit sur le fichier remove_watermark.bat

Sélectionner « Exécuter en tant qu’administrateur »

Attendre la fin de l’exécution

Redémarrer l’ordinateur (recommandé)

Après le redémarrage, le watermark doit avoir disparu.

Important

Le masquage est visuel uniquement

Après une grosse mise à jour Windows, il peut être nécessaire de relancer le script

Aucun risque pour le système

Annulation

Pour rétablir l’affichage par défaut, supprimer la clé PaintDesktopVersion ou la remettre à 1, puis redémarrer.
